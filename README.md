Create GitHub repository for Atcoder codes.
```
cd codes
git init
git remote add origin (your reopsitory)
(e.g. git remote add origin git@github.com:kult0922/AtCoderArchive.git)
```

Input your user name and code extension
```
{
  "user": "Kurt_",
  "extension": "cpp"
}
```

Run 
```
Python3 abc290
```
